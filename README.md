## About The Project

Vhiweb take home assignement for creating a simple single page application (SPA) by displaying data from https://reqres.in/.

## Built With

- [React JS](https://reactjs.org/)
- [SASS](https://sass-lang.com/)

## Requirements

1. Ensure user experience
2. Ensure performance
3. Clean code

## Installation

1. Clone Repository

```sh
git clone https://github.com/emhaarifin/test-vhiweb.git
```

2. Go To Folder Repo

```sh
cd test-vhiweb
```

3. Install Module

```sh
npm install
or
yarn
```

4. Start website

```sh
npm start
or
yarn start
```
